# Operations Analyst Portfolio Website

This folder contains a static portfolio site built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main website page
- `styles.css` — visual styling for the site
- `script.js` — mobile navigation toggle
- `Samantha Eichelberger Professional Portfolio.pdf` — uploaded PDF portfolio linked from the home page
- `README.md` — this file

## How to open the website

### Option 1: Open directly in your browser
1. Open File Explorer and navigate to `c:\Users\bones\OneDrive\Desktop\Portfolio website`
2. Double-click `index.html`
3. The page will open in your default browser

### Option 2: Open from VS Code
1. Open the folder in VS Code
2. Right-click `index.html`
3. Select `Open with Live Server` if you have the Live Server extension installed

### Option 3: Use a simple local server
If you want to view the site on a local development server, use a command prompt or PowerShell from the folder and run:

- With Python 3 installed:
  - `python -m http.server 8000`
- Then open `http://localhost:8000`
